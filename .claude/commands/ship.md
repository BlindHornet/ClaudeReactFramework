Prepare this work for commit:

1. Run `npm run lint` — fix any errors
2. Run `npm run build` — confirm no type errors
3. Run `npm run test` — all tests must pass
4. Review the diff — make sure nothing unintended is staged
5. Create a conventional commit: feat/fix/refactor + clear message
6. Update tasks/todo.md — mark completed items

Do not ship if tests fail.
